    ¿Que es MX-Bot?

    Nuestro bot tiene la capacidad de ser un bot interactivo e entretenido como tambien ser un bot de negocios.

    ✓

    Rendimiento

    Un bot estable, rapido y facil de usar.

    ✓

    Comandos

    Incluye comandos extra y funciones avanzadas.

    ✓

    Personalización total

    Adapta el bot a tu manera con sus estilos unicos en el chat.

    Pequeña Guía

    /menu Probar

    Muestra una lista de menu con comandos disponibles para usarse.

    /tyc Probar

    Muestra los terminos y condiciones de uso para evitar ciertos problemas.

    /stat Probar

    Ver el estado del bot, todo el detalle de funcionamiento en este comando.

    /bot reglas Probar

    Muestra las reglas establecidas en el bot y lo que no esta admitido hacer.

    /g-data Probar

    Muestra opciones de configuración para grupos, el comando solo puede usarse en grupos.

    /logo Probar

    Crea logos personalizados para tu perfil o otra cosa, este comando tiene sus variantes también.

    /play probar

    Descarga videos o musicas con este comando, tambien encontraras otras descargas en el menu de descarga.

    /imagina Probar

    Genera imágenes a base de texto, podras generar los que quieras sin limites.

    /chatgpt Probar

    Usa este comando para hablar con ChatGPT, sin limites y gratuito.

    /report Probar

    Reporta los comandos erroneos para que podamos brindarte una experiencia fluida.
